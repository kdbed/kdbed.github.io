The Basics

Hello

Begin with a simple hello world program kurt2018get:

{{< highlight haskell “linenos=table, linenostart=1” >}} –hello.hs : first haskell file main = do print “Hello World!” {{< /highlight >}}

Compile the code by passing hello.hs to GHC as an argument:

{{< highlight shell “linenos=table, linenostart=1” >}} ghc hello.hs {{< /highlight >}}

Thus GHC has created three files: hello, the binary executable, along with hello.hi and hello.o. The binary can be executed from the command line:

{{< highlight shell “linenos=table, linenostart=1” >}} ./hello {{< /highlight >}}

"Hello World!"

Messy Main

To gain some intuition for the structure of Haskell programs, we introduce messyMain, which defies the good practice of writing modular code:

{{< highlight haskell “linenos=table, linenostart=1” >}} –messyMain.hs : an example to fix main :: IO() main = do print “Who is the email for?” recipient <- getLine print “What is the Title?” title <- getLine print “Who is the Author?” author <- getLine print (“Dear” ++ recipient ++ “, ” ++ “Thanks for buying” ++ title ++ “, ” ++ author ) {{< /highlight >}}
